# GitHub Sources For Dialogue Data

Muc tieu cua file nay la ghi lai cac nguon GitHub co the dung de tham khao cau truc dataset hoi thoai, QA hoac chatbot cho phan AI chatbox.

## 1. Vista

- Repo: https://github.com/Oztobuzz/Vista
- Mo ta: dataset hoi thoai tieng Viet quy mo lon, huong multimodal.
- Cach dung: tham khao format hoi thoai va cach to chuc mau dialogue. Khong can lay het vi domain cua ban la cafe.

## 2. conversation_clustering

- Repo: https://github.com/trieuntu/conversation_clustering
- Mo ta: xu ly hoi thoai tieng Viet tu Facebook Page de tao training data cho chatbot.
- Cach dung: phu hop de hoc cach lam sach va nhom cac cau hoi khach hang thanh intent.

## 3. VietnameseDatasets

- Repo: https://github.com/kietnv/VietnameseDatasets
- Mo ta: tong hop nhieu benchmark dataset tieng Viet nhu UIT-ViQuAD, ViNewsQA, UIT-VSFC.
- Cach dung: tham khao cho bai toan QA va cau truc du lieu hoi dap, dac biet neu ban muon mo rong sang FAQ retrieval.

## 4. Vietnamese-RAG-based-QA-System

- Repo: https://github.com/nhmh2612/Vietnamese-RAG-based-QA-System
- Mo ta: demo chatbot QA tieng Viet su dung du lieu UIT-ViQuAD.
- Cach dung: tham khao cach noi du lieu FAQ/QA vao chatbot sau khi phan loai intent.

## 5. TelecomDomainFineTuning

- Repo: https://github.com/knijesh/TelecomDomainFineTuning
- Mo ta: sinh hoi thoai customer-support theo domain rieng.
- Cach dung: tham khao cach viet hoi thoai nhieu luot co y dinh ro rang. Repo nay la tieng Anh, nhung format rat hop de tao du lieu seed cho quan cafe.

## Cach ap dung vao du an nay

1. Khong copy nguyen xi dataset ngoai vao chatbot cafe.
2. Chi dung chung de hoc format hoi thoai, format multi-turn va cach dat nhan intent.
3. Dataset train thuc te van nen tu tao tu menu va nghiep vu cua quan.
